关于线性代数的直观理解

关于线性代数的直观理解

线性变换的概念

  • 什么叫线性变换，就是空间通过伸缩等变成平行等距的，原点不变的空间。假设是二维的空间，就是很多等距平行的正方形网格。假设是三维的就是很多等距平行的立方体模型。

  • 线性变换的应用：

    • 重积分坐标变换要乘雅克比行列式，因为行列式本质就是由基向量构成的矩阵的面积，缩放，让每一个小的图形尽可能地变小，去贴合不规则图形的形状，从而更好地求得不规则图形的面积。行列式的值为0就相当于将这个空间降维了。这一点就好比是有一块很大的拼图，不规则形状，假设打上了马赛克，即用大一点的正方形组成拼接的，但是可能会有地方超出来。假设改成菱形呢，通过矩形变换去拟合，或者变成更小一点的正方形去拟合呢？说不定会更好。

关于行列式的直观理解

  • 行列式的值如果为负数，那么其绝对值的值是缩放的比例，负号表示空间翻转，就像一张纸一样。
  • 二阶行列式其实是一个二维平面四边形的面积；三阶行列式其实是一个平行六面体的体积。

关于秩、线性空间的直观理解

  • 秩是用来描述空间经过线性变换以后空间的维数，是一维的还是二维的，还是三维的。

  • 所有可能的输出像两个Av构成的集合称为A的列空间。矩阵的列所张成的空间。

关于线性方程和它的解

  • 一个线性方程，可以通过逆变换来求方程组的解，即还原。

  • 如果一个三维矩阵降到二维，说明，有一个直线上的所有向量被压缩成了原点，如果一个三维矩阵降到了一维，说明，有一整个平面的所有向量被压缩成了原点。以此列推。这些被压缩在原点里面的就构成了零空间

  • 列空间的概念在于什么时候有解

    • 列空间就是矩阵的列所张成的空间。秩的更加精确的定义是列空间的维数。当秩达到最大值时，意味着秩与列数相等。零向量一定包含在列空间当中，因为线性变换必须保持原点位置不变。对于一个满秩变换来说，唯一能在变换后落在原点的就是零向量自身。但是对于一个非满秩的矩阵来说，它将空间压缩到一个更低的维度上（一系列的向量在变换后成为零向量）。二维线性变换将空间压缩到一条直线上，沿着直线上不同方向的所有向量就被压缩到原点。三维线性变换将空间压缩到一个平面上，同样会有一整条线上的向量在变换后落在原点。如果三维线性变换将空间压缩到一条直线上，将会有一整个平面上的向量在变换后落在原点。在变换后落在原点的向量集合，被称为所选矩阵的“零空间”或者“核”，变换后一些向量落在零向量上，从意义上来说，“零空间”就是这些向量张成的空间。

  • 非方阵有点难理解，方阵都是由一个空间转换到对应的维度。3x2的矩阵的意思是：2代表两个基向量，3代表每一个基向量变换后都用三个独立的坐标来描述。换句话说，就是一个二维平面放到三维空间里去描述。 2x3矩阵的意思是，有3个基向量，每一个基向量用2个坐标来表示，它还是满秩的，因为列空间的维数与输入空间的维数相等。

关于向量的数量积和叉积

  • 向量的数量积和点积

    • 向量的数量积就是类似两个2x1的矩阵相乘，向量都是立着的，1x2矩阵就相当于二维平面中的一维数轴，然后变换2x1的坐标点到数轴上。

    • 1x2向量[2 3]与二维矩阵（立着的）[2 3] 之间的关系，二维空间压缩成一根一维的数轴。

  • 点积：与投影有关。

    • 直观的理解：把两个向量看成是一个矩阵对另一个向量的线性变换。
    • 描述投影变换的1x2矩阵的两列，就分别是u帽的两个坐标！！！！！牛逼！！！！1x2矩阵是用来描述变换的。

  • 如何解释投影变换和点积是一样的方法？

    • 投影先把二维上的基向量i,j变成了数轴上的ux，uy，这个过程就是变换，以i，j为基向量的向量变换都要进行乘积

  • 注意点：点积是数，矩阵乘法是向量，两者仅数值相等，不完全相等

  • 叉积的方向根据右手定则，大拇指的方向就是叉积的方向。空间理解就是一个平行六面体，一个固定的底面积，乘以高，所谓的高就是（x,y,z）在垂直于底面方向上的投影。又根据对偶性，向量P就等价于v,w且值等于平行四边形面积的向量。这个等价于向量P（垂直于v,w且值等于平行四边形面积的向量）与（x,y,z）向量点乘。

关于逆变换和特征向量、特征值

  • 我们的基变换的逆x线性变换x我们的基变换 x She描述的向量=转移，视角的转化。先以我们的角度看，再以She的角度看

  • 找特征向量本质上是找旋转轴，一个空间经过变换以后可能仍然会有向量在自己原来张成的空间中（即原来的直线上），只是被缩放了，所以缩放的比例就是特征值。那么描述这一特征，就可以写成Av=λv，左边是两个向量相乘，右边是数乘，为了将数乘转化为向量，可以将λ看成是λ乘以一个三阶单位矩阵（即λI），那么Av=(λI)v可以转换成(A-λI)v=o向量，当且仅当矩阵代表的变换将空间降维时，才会存在一个非零向量，是的矩阵和它的乘积为零向量。即行列式为0。

  • 如果要计算一个矩阵100次幂，可以先转换到不变的特征基，再进行计算。
